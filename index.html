About Me
Hello, my name is Ethan Mo. I graduated from University of Washington Bothell campus, with a bachelor's degree in Computer Science and Software Engineering.

I'm always seeking opportunities to learn new technologies and improve my skill set.

Please take a look at some of the projects I've worked on!
Projects

Touching Function

Touching Function, created in collaboration with CRCS (Cross Reality Collaboration Sandbox) Research Group, is a 3-D graphing tool/game that is intended to assist math teachers and learners in the remote learning environment.

Relavant technologies: C#, Unity, Wolfram Alpha, OpenXR, NuGet, GitHub.

Archery Simulator

A archery simulation game developed using Unity and C#.

The game is intended to demonstrate high level of knowledge in math concepts including algebra, trigonometry and calculus, as the physical interactions between objects, and the shaders are all hand-coded.

ReadRight

ReadRight is an online web-based service that uses Speech-to-Text APIs, a microphone, and a unique analytical algorithm to show the user a paragraph of text in English and analyzes the way they are speaking in order to provide tips on grammatical or phonetical mistakes as well as improvements on overall diction.

Practice Engine

A practice environment for users to learn and improve their JavaScript skills.

The application offers a list of JavaScript questions, and uses CodeMirror library for users to input their codes. The codes are examed with preset test cases.

A progress tracker and a login system is also implemented.

TF Club website

The official website for Trickfire Robotics Club of University of Washington.

Written in HTML, CSS and JavaScript. Google calendar API is used to display events.

Word Guess Game

A Pokemon themed, Hangman style mini game, written in HTML, CSS and Javascript.

Rain On Your Parade

A webapp to search for local events using location and/or date, and check for weather in that area for the event's date, using Google Maps and EventBrite API.
